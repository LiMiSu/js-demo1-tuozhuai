# js-demo1-tuozhuai

如果要设置物体拖拽，那么必须使用三个事件，并且这三个事件的使用顺序不能颠倒。
```

1.onmousedown：鼠标按下事件
2.onmousemove：鼠标移动事件
3.onmouseup：鼠标抬起事件
```
```
重点：

1、一定要绝对定位，脱离文档流才可以移动,才有方向值。

2、绑定拖拽的元素，移动和鼠标松开是对document的绑定，因为移动的是整个div。

3、点击：a= 获取当前鼠标坐标、b =div距浏览器距离、c = 鼠标在div内部距离=a-b。

    移动：通过  a - c 建立鼠标与div的关系，防止鼠标超出div。
```
    
```
基本思路1：

拖拽状态 = 0鼠标在元素上按下的时候{  

  拖拽状态 = 1  

  记录下鼠标的x和y坐标  

  记录下元素的x和y坐标  

  } 

 鼠标在元素上移动的时候{  

  如果拖拽状态是0就什么也不做。  

  如果拖拽状态是1，那么  

  元素y = 现在鼠标y - 原来鼠标y + 原来元素y  

  元素x = 现在鼠标x - 原来鼠标x + 原来元素x  

  }   

 鼠标在任何时候放开的时候{  

  拖拽状态 = 0  

}
```
```
基本思路2：
拖拽状态 = 0鼠标在元素上按下的时候{  

  拖拽状态 = 1  

  记录下鼠标与div左边的距离

  记录下鼠标与div上边的距离  

  } 

 鼠标在元素上移动的时候{  

  如果拖拽状态是0就什么也不做。  

  如果拖拽状态是1，那么  

  元素y = 现在鼠标y - 鼠标与div上边的距离  

  元素x = 现在鼠标x - 鼠标与div左边的距离 

  }   

 鼠标在任何时候放开的时候{  

  拖拽状态 = 0  

}

```
